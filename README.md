# (not so) Secret Notifications - COMP 599 A1

Coding examples for the **"(not so) Secret Notifications"** presentation. The examples are split into two applications, each of which consists of an independent Android Studio project.

## [`listener_app`](./listener_app)

Listens to notifications on the device. Template modified from [Chagall/notification-listener-service-example](https://github.com/Chagall/notification-listener-service-example).

## [`notifier_app`](./notifier_app)

Posts notifications for the listener app to capture.
